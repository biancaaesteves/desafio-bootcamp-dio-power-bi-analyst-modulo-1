# 📊 Desafio Power BI Analyst — Criação de Relatório

## 📌 Descrição do Projeto

Este projeto foi desenvolvido como parte da **Formação Power BI Analyst da DIO (Digital Innovation One)**, com o objetivo de aplicar na prática os conceitos de criação de relatórios e visualizações no Power BI.

O desafio consiste na construção de um relatório com base em uma base de dados disponibilizada no curso, replicando páginas existentes e desenvolvendo uma nova página com visuais personalizados.

---

## 🎯 Objetivo

* Praticar a criação de dashboards no Power BI
* Desenvolver habilidade na construção de visuais
* Trabalhar com organização e clareza na apresentação dos dados
* Consolidar conceitos iniciais de BI e storytelling com dados

---

## 📂 Base de Dados

Os dados utilizados neste projeto foram disponibilizados no repositório oficial do desafio:

🔗 https://github.com/julianazanelatto/power_bi_analyst

---

## 📊 Estrutura do Relatório

O relatório é composto por **3 páginas**:

### 🔹 Página 1 e 2

* Replicação das páginas apresentadas durante o curso
* Foco em entendimento da estrutura dos visuais
* Aplicação de boas práticas de organização

### 🔹 Página 3 (Desenvolvida no desafio)

Foram criados os seguintes visuais:

* 🌍 **Mapa 1:** Soma de *Sales* e *Units Sold* por país
* 🌍 **Mapa 2:** Soma de *Profit* por país
* 🥧 **Gráfico de Pizza:** Lucro por segmento

---

## 🛠️ Tecnologias Utilizadas

* Power BI Desktop
* Modelagem de dados
* Criação de visuais interativos

---

## ✨ Diferenciais Aplicados

* Organização estratégica dos visuais no relatório
* Renomeação dos gráficos para melhor entendimento
* Uso adequado de campos em **tooltips (dicas de ferramenta)**
* Estrutura visual voltada para fácil leitura e interpretação
