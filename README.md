# desafio-bootcamp-dio-power-bi-analyst-modulo-1
Projeto desenvolvido durante a Formação Power BI Analyst (DIO), com foco na criação de relatórios e visualizações no Power BI. O desafio consistiu na replicação de páginas existentes e na construção de uma nova página com mapas e gráfico de pizza, explorando métricas como sales, profit e segmentação de dados.
